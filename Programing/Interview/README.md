# Interview

##❯ FizBuzz

  Really stupid interview test to prove that interviewee is aware of modulo operator.

  Example:
```javascript
var i;

for (i = 1; i < 101; i = i + 1) {

    if ((i % 3 === 0) && (i % 5 === 0)) {
        listItem = "fizzbuzz";
    }
    else if (i % 3 === 0) {
        listItem = "fizz";
    }
    else if (i % 5 === 0) {
        listItem = "buzz";
    }
    else {
        listItem = i;
    }

    console.log(listItem);
}
```
